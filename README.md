# MKBrain  
Социально-образовательная платформа

Полноценная веб-платформа, объединяющая образовательный функционал (тесты, статистика, оценки) и социальные возможности (чаты, классы, новости).  
Проект разработан в рамках индивидуального проекта (10 класс)

## Основная идея проекта

Большинство образовательных платформ предоставляют инструменты обучения,  
но **не имеют нормальной системы коммуникации между учителем и учениками**.  
При этом социальные платформы не предоставляют функций проверки знаний.

**MKBrain решает эту проблему:**  
на одной платформе собраны тесты, классы, чаты, статистика, новости, управление доступами и роли пользователей.

---

## Функциональность

### 1. Система ролей
- **Свободный пользователь** — базовые функции, ограниченный конструктор тестов  
- **Ученик** — тесты, статистика, классы  
- **Учитель** — классы, оценки, полный конструктор тестов  
- **Директор организации** — управление школой, новостями, классами  

### 2. Образовательные инструменты
- Конструктор тестов любого типа  
- Настройка метода оценивания  
- Приватные/публичные тесты  
- Автоматическая проверка ответов  

### 3. Социальные функции
- Чаты (личные и групповые)  
- Новости школы и классов  
- Домашние задания  
- Друзья, заявки, уведомления  

### 4. Статистика
- % выполнения теста  
- Количество правильных/неправильных ответов  
- Общий рейтинг ученика  
- Статистика по классам

## Технологии и архитектура

### **Frontend**
- HTML  
- CSS (Bootstrap, FontAwesome)  
- JavaScript  
- AJAX (динамическая подгрузка данных)  
- Графики: **MyChart.js**

### **Backend**
- PHP  
- MySQL  
- Собственная MVC-архитектура  
- Работа через роутер `.htaccess`

### **Структура проекта**
- `/controllers` — контроллеры  
- `/models` — работа с БД  
- `/views` — шаблоны, страницы  
- `/assets` — css/js/img  
- `config/constants.php` — глобальные переменные, пути, настройки  
- `index.php` — точка входа + роутер

## Как запустить локально

```sh
git clone https://github.com/Dany21mk/mkbrain
```
1. Установить XAMPP или OpenServer
2. Создать базу данных mkbrain
3. Импортировать дамп
4. Настроить файл config/constants.php
5. Запустить Apache и MySQL
